## Danh sách Lab
### 8051
1. **Lab 1:** ACQUAINTANCE WITH ***PROTEUS*** AND THE ***8051*** MICROCONTROLLER FAMILY
  - Heart-LED circuit consisting of 32 LEDs
2. **Lab 2:** COMMUNICATION WITH ***7-SEGMENT LED*** AND ***TIMER***
  - Using the Timer, design a clock circuit with 24h format with the initial time set in the source code
3. **Lab 3:** USING ***INTERRUPT***
  - Design a Sport clock circuit using AT89C51/AT89C52 with 4 7-Segment LED modules
  - Counting range: 00.00 s → 99.99 s with 1% second accuracy
  - Control:
    - Button A: Pause/Resume stopwatch
    - Button B: Reset the stopwatch
    - Button C: Increase seconds by 1
    - Button D: Decrease seconds by 1
4. **Lab 4:** ***USING UART***
  - Disply calculations and results on an LCD that receives data via UART
### 8086
5. **Lab 5:** ADDITION OF TWO ***32-BIT NUMBERS*** ON THE 8086 PROCESSOR
  - Subtract 2 32-bit numbers
6. **Lab 5:** IO PROCESSING, CALCULATION AND MEMORY ON THE 8086 MICROPROCESSOR
  - Print the first N Fibonacci numbers
